# TableOfContents Extension

Generates a table of contents from the headings in a document at the `@toc`
marker position. Heading IDs are auto-generated and linked automatically.

## Basic Usage

```php
use Alto\CommonMark\Extension\TableOfContents\TableOfContentsExtension;
use League\CommonMark\Environment\Environment;
use League\CommonMark\MarkdownConverter;

$environment = new Environment();
$environment->addExtension(new TableOfContentsExtension());

$converter = new MarkdownConverter($environment);
```

```markdown
@toc

# Introduction

## Getting Started

### Installation

## Configuration
```

```html

<div class="table-of-contents" id="toc">
    <ul>
        <li><a href="#introduction">Introduction</a></li>
        <li>
            <ul>
                <li><a href="#getting-started">Getting Started</a></li>
                <li>
                    <ul>
                        <li><a href="#installation">Installation</a></li>
                    </ul>
                </li>
                <li><a href="#configuration">Configuration</a></li>
            </ul>
        </li>
    </ul>
</div>

<h1 id="introduction">Introduction</h1>
...
```

## Syntax

```markdown
@toc
@toc {min: 2, max: 3}
@toc {min: 2, max: 4, ordered: true}
```

The directive must be at column 0 (no indentation). Multiple `@toc` directives
in the same document each generate a separate TOC.

### Per-directive options

| Option    | Type | Example           | Description                            |
|-----------|------|-------------------|----------------------------------------|
| `min`     | int  | `{min: 2}`        | Minimum heading level to include (1–6) |
| `max`     | int  | `{max: 3}`        | Maximum heading level to include (1–6) |
| `ordered` | bool | `{ordered: true}` | Use `<ol>` instead of `<ul>`           |

## Constructor

```php
new TableOfContentsExtension(array $config = [])
```

| Config key  | Type         | Default               | Description                              |
|-------------|--------------|-----------------------|------------------------------------------|
| `min_level` | int          | `1`                   | Default minimum heading level            |
| `max_level` | int          | `6`                   | Default maximum heading level            |
| `style`     | string       | `'bullet'`            | `'bullet'` (ul) or `'ordered'` (ol)      |
| `class`     | string       | `'table-of-contents'` | CSS class for the wrapper `<div>`        |
| `id`        | string       | `'toc'`               | `id` attribute for the wrapper `<div>`   |
| `title`     | string\|null | `null`                | Optional heading rendered above the list |
| `marker`    | string       | `'@toc'`              | Recognition marker (customisable)        |

## Output Examples

### Filtered levels

```markdown
@toc {min: 2, max: 3}
```

Only h2 and h3 headings appear in the TOC.

### Ordered list

```markdown
@toc {ordered: true}
```

```html
<div class="table-of-contents" id="toc">
  <ol>
    <li><a href="#section-1">Section 1</a></li>
    <li><a href="#section-2">Section 2</a></li>
  </ol>
</div>
```

### With title (global config)

```php
new TableOfContentsExtension(['title' => 'Contents'])
```

```html

<div class="table-of-contents" id="toc">
    <h2>Contents</h2>
    <ul>...</ul>
</div>
```

## Heading ID Generation

Heading IDs are generated by slugifying the heading text:

- Lowercase: `My Document` → `my-document`
- Non-alphanumeric characters replaced with hyphens: `Section 1.1` →
  `section-1-1`
- Leading/trailing hyphens trimmed
